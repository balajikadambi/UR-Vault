## Documentation

### Why UR-Vault?
- UR-Vault has been built around maintaining privacy and security with transparency. 
- UR-Vault stores all the data on your device. The data is stored on the macOS/iOS keychain. The syncing of data between your apple devices uses Apple [iCloud Keychain sync](https://support.apple.com/en-in/109016). 
For more details, please refer [iCloud Keychain security overview](https://support.apple.com/en-in/guide/security/sec1c89c6f3b/web)
- No data can be viewed on the device without your authentication.
- No password or numeric pin is stored. It is re-generated on demand provided the correct secret is provided and device authentication is successful.
- Only the password or numeric pin generation settings are stored for every site which cannot be viewed once generated.
- The export of data exports the password or numeric pin generation settings in encryted format with user provided encryption key.
- No personal data is collected by UR-Vault.
